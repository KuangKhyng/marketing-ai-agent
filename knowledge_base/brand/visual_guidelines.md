# Visual Guidelines — TửViOnline

## Color Palette
- Primary: Deep Purple (#4A1A8A) — mystery, spirituality
- Secondary: Gold (#D4A847) — premium, wisdom
- Accent: Soft Teal (#5BB5A2) — calm, trust
- Background: Dark Navy (#0F1729) — night sky feel
- Text: Off-white (#F5F0E8) — warm, easy on eyes

## Typography
- Headings: Rounded, modern serif (e.g., Lora, Playfair Display)
- Body: Clean sans-serif (e.g., Inter, Be Vietnam Pro)
- Accent text: Handwritten feel for quotes/highlights

## Visual Style
- Minimalist với cosmic/celestial elements
- Gradient backgrounds (purple → navy)
- Star/constellation patterns as subtle textures
- Avoid: quá many mystical symbols, hình ảnh mê tín (bát quái, bùa chú, etc.)

## Photography/Illustration
- Prefer: illustrations over stock photos
- Style: modern, flat-ish, với cosmic/night sky theme
- People: nếu dùng, nên là young, diverse, contemplative mood
- Avoid: crystal balls, tarot cards, overly mystical imagery

## Social Media Templates
- Instagram carousel: dark background, gold accent text, 1 key point per slide
- Facebook post: lighter background variant, readable font size
- TikTok: text overlays with slight glow effect, dark theme
